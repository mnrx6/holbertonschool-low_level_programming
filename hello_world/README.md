# C - Hello, World

This project introduces the basics of the C programming language.

## Learning Objectives

- What happens when you compile a C program
- The compilation process
- What is the C preprocessor
- How to compile a C program using gcc
- The difference between source files, object files, and executable files
- Basic use of gcc compiler options
