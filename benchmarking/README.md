# Green Tech Efficiency & Benchmarking Lab

## Description
This project measures the execution time of a C program using `clock()` to establish a baseline for performance benchmarking.

## Compilation

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 -Wno-long-long baseline_loop.c -o baseline_loop
```

## Usage

```bash
./baseline_loop
```

Run the program three consecutive times and save the output in `baseline_loop-metrics.md`.
