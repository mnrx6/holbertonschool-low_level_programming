# Simple Calculator

## Description

This project is a simple terminal-based calculator written in C. It displays a menu that allows the user to choose an arithmetic operation. In this first version, the program displays the menu, reads the user's choice, and exits when the user selects the quit option.

## Compilation

Compile the program using:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator
```

## Usage

Run the program with:

```bash
./calculator
```

## Menu

- 1) Add
- 2) Subtract
- 3) Multiply
- 4) Divide
- 0) Quit

## Current Features

- Displays a calculator menu.
- Reads the user's menu choice.
- Exits the program when the user selects `0`.
